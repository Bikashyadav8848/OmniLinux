# OmniLinux Brand Style Guide

## Logo
The OmniLinux logo represents the infinite power of open-source computing.

| OmniCyan | `#56b6c2` | Terminal, code, tech elements |

## Typography
- **Headings**: Inter (weight: 600-700)
- **Body**: Inter (weight: 400)
- **Code/Terminal**: JetBrains Mono Nerd Font
- **Fallback**: system-ui, sans-serif

## Design Principles
1. **Glassmorphism** — frosted glass effects with blur
2. **Dark by default** — easy on the eyes, modern look
3. **Rounded corners** — 8px for elements, 12px for cards
4. **Smooth animations** — 200-300ms transitions
5. **Consistent spacing** — 8px grid system
6. **Vibrant accents** — colors should pop against dark backgrounds

# OmniLinux Plymouth Theme

Boot splash screen for OmniLinux.

## Files

- omnilinux.plymouth - Theme descriptor
- omnilinux.script - Animation script

## Installation

Automatically applied during ISO build via hooks.

## Requirements

- Plymouth 0.9.0+

## Preview

Uses OmniLinux colors:
- Primary: #3D7EFF (OmniBlue)
- Secondary: #7C3AED (OmniPurple)
- Background: #0D1117 (OmniDark)

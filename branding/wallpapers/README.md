# OmniLinux Default Wallpapers

OmniLinux comes with curated wallpapers.

## Contents

Place wallpapers here before build:
- 1920x1080 (Full HD)
- 2560x1440 (2K)
- 3840x2160 (4K)

## Default

Uses Ubuntu default wallpapers (Warty) as fallback.

## Adding Custom Wallpapers

1. Add PNG/JPG to this directory
2. Add to config/hooks/02-theming.hook.chroot

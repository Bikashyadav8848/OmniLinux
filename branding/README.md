# OmniLinux Branding Assets

## Colors
- OmniBlue: #3D7EFF
- OmniDark: #0D1117
- OmniLight: #F0F4FF
- OmniPurple: #7C3AED
- OmniGreen: #10B981

## Contents
This directory contains visual branding assets for OmniLinux:
- GRUB theme files
- Plymouth boot splash themes
- Custom wallpapers
- Icon packs (OmniIcons)

The actual theme files need to be generated or added here.
For GRUB theming, see: https://github.com/szymons/-grub2-themes
For Plymouth themes, see: https://github.com/AdelKhalid/plymouth-themes

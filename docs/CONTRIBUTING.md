# Contributing to OmniLinux

## Ways to Contribute

1. **Testing**: Test the ISO in VMs, report bugs
2. **Packaging**: Improve package lists
3. **Theming**: Create OmniLinux branding
4. **Documentation**: Improve guides

## Development Flow

1. Fork the repo
2. Create feature branch
3. Make changes
4. Test with live-build
5. Submit PR

## Package List Guidelines

- Use only official Ubuntu 24.04 packages
- Avoid packages requiring external PPAs
- Test packages exist before adding

## Theme Guidelines

- Use OmniLinux color palette
- Keep themes lightweight
- Support light/dark modes

## Communication

- Report issues on GitHub
- Join community (link in docs)
